### CorporateProductParametersService
Types:
- `ProductParameter = { id, name, description, embedding? }`
- `CrawlEmbeddingResult = { id, content, distance, metadata? }`
- `EvidenceSet = { productId, productName, evidence: CrawlEmbeddingResult[] }`
- `ProductSignal = { productId, productName, signals, summary, confidence, rawEvidenceCount }`
- `SignalExtractionResult = { companyId, locale, productSignals, timestamp }`
- `EnvironmentDetails = Record<string, ProductParameter>`

Functions:
- `parseCorporateProductParameters(content): EnvironmentDetails`
- `parseCorporateProductParametersFromFile(filePath): EnvironmentDetails`
- `parseCorporateProductParametersByLocale(locale): EnvironmentDetails`
- `embedProductParameters(locale): Promise<EnvironmentDetails>`
- `retrieveByProductParameters(companyIdOrCode, locale, topK): Promise<EvidenceSet[]>`
- `extractSignalsFromEvidence(evidenceSets, companyId, locale): Promise<SignalExtractionResult>`
- `retrieveAndExtractSignals(companyIdOrCode, locale, topK): Promise<SignalExtractionResult>`
